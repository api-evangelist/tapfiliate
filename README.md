# Tapfiliate (tapfiliate)

Tapfiliate is an affiliate tracking and management platform that enables businesses to quickly launch and grow affiliate, referral, influencer, and partner programs. With over 69,500 customers across e-commerce and SaaS industries, Tapfiliate provides transparent flat-fee pricing with no percentage taken from sales. The platform offers a REST API (v1.6) for creating affiliate programs, managing affiliates, tracking conversions, and handling commission payouts, along with 30+ pre-built integrations and SDKs for PHP, JavaScript, Python, Ruby, Go, Swift, Java, and more.

## APIs

- **Tapfiliate REST API** — REST API for managing affiliate programs, affiliates, conversions, commissions, customers, payments, and click tracking. Base URL: https://api.tapfiliate.com/1.6/. Authentication via X-Api-Key header.

## Plans / Rate Limits / FinOps

- [Plans & Pricing](plans/tapfiliate-plans-pricing.yml) — Three tiers: Launch ($89/mo), Scale ($179/mo), Enterprise (custom). Annual billing saves ~17%. No percentage fees.
- [Rate Limits](rate-limits/tapfiliate-rate-limits.yml) — Limits communicated via X-Ratelimit-Limit, X-Ratelimit-Remaining, and X-Ratelimit-Reset response headers. Pagination defaults to 25 items per page.
- [FinOps](finops/tapfiliate-finops.yml) — Flat subscription cost model. Key cost drivers are monthly click and conversion volume caps per plan tier.
